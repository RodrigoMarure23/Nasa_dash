🚀 NASA Asteroid Radar Dashboard

Dashboard interactivo para visualizar asteroides cercanos a la Tierra (NEO) usando datos reales de la NASA.

El proyecto combina visualización de datos, gráficos interactivos, animaciones y renderizado 3D para crear una experiencia tipo centro de control espacial.

✨ Características
🛰 Monitoreo de Asteroides

Visualización de datos de asteroides cercanos a la Tierra usando la API de la NASA:

    -Velocidad

    -Diámetro estimado

    -Distancia mínima a la Tierra

    -Clasificación de peligro

🌍 Radar Interactivo

    Simulación visual tipo radar espacial.

    Funciones:

    Asteroides orbitando alrededor de la Tierra

    Identificación de asteroides peligrosos

    Tooltips informativos

    Navegación accesible

🪐 Visualización 3D

    Renderizado 3D con Three.js:

    Modelo procedural de asteroide

    Rotación dinámica

    Entorno espacial con partículas

    Resaltado de asteroides peligrosos

📊 Gráficas de Datos

    Visualización usando Recharts:

    Asteroides detectados por día

    Comparación entre peligrosos y no peligrosos

    Tooltips personalizados

    Layout responsive

🎛 Animaciones de Interfaz

    Animaciones suaves con Framer Motion:

    Transiciones entre componentes

    Animaciones de carga

    Interacciones del radar

🌐 Internacionalización

Sistema de traducción integrado:

    Idiomas disponibles:

        🇪🇸 Español

        🇺🇸 Inglés

Implementado con hooks personalizados en React.

⚙️ Instalación

    Clonar el repositorio:

        git clone https://github.com/TU-USUARIO/asteroid-dashboard.git

    Entrar al proyecto:

        cd asteroid-dashboard

    Instalar dependencias:

        npm install

    Ejecutar en desarrollo:

        npm run dev

🏗 Build para Producción
    npm run build

Previsualizar build:

    npm run preview
